# Wallie

## Vision

Wallie is a personal AI and knowledge platform designed to become a long-term second brain.

The system stores knowledge, technical decisions, project documentation, engineering notes, and future sensor data in a structured and searchable format.

## Current Platform

- Raspberry Pi 4
- Ubuntu Server
- Docker
- Portainer
- Open WebUI
- Tailscale

## Long-Term Goals

- Personal knowledge management
- Engineering knowledge base
- Project documentation
- Sensor integration
- Automation platform
- AI-assisted reasoning
