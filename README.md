# WallieVault

WallieVault es la base de conocimiento principal del proyecto Wallie.

Este repositorio almacena notas en formato Markdown compatibles con Obsidian.

## Estructura

- `00_Inbox`: ideas rápidas y notas pendientes de organizar.
- `01_Projects`: proyectos activos como Wallie, Orion y laboratorios.
- `02_Knowledge`: conocimiento permanente por área.
- `03_References`: documentación externa, enlaces, datasheets y recursos.
- `04_Daily`: bitácoras diarias y seguimiento.
- `05_Templates`: plantillas reutilizables.

## Filosofía

El conocimiento debe vivir primero como archivos Markdown simples, versionados con Git y fáciles de leer por humanos y por sistemas de IA.
