# Use Markdown As Canonical Format

Date: 2026-06-14

## Decision

All long-term knowledge stored by Wallie will be saved as Markdown files.

## Reason

- Open format
- Tool independent
- Easy to parse by LLMs
- Future-proof

## Consequences

### Positive

- Portability
- Simplicity
- Easy version control

### Negative

- Requires file organization standards

## Status

Approved
