# Replace Oracle With Tailscale

Date: 2026-06-02

## Decision

Replace Oracle Cloud VM and WireGuard with Tailscale as the primary remote access solution.

## Reason

- Lower maintenance
- No dependency on Oracle Free Tier
- Easier device onboarding
- Better user experience

## Consequences

### Positive

- Simplified networking
- Reduced infrastructure complexity
- Direct device-to-device connectivity

### Negative

- Dependency on Tailscale control plane

## Status

Approved
